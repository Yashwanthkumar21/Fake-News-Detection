
# Fake News Detection using BERT

## 📌 Problem
Detect whether news is fake or real using NLP.

## ⚙️ Approach
- Data preprocessing
- Tokenization using BERT
- Model training

## 🤖 Model
bert-base-uncased

## 📊 Results
The model was trained and evaluated using standard metrics like accuracy, precision, recall, and F1-score.

Due to computational limitations, the model was trained with fewer epochs.

## 📚 Learnings
- NLP basics
- Transformer models
- Model evaluation
